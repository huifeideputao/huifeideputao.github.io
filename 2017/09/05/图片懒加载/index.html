jQuery Lazy Load实现图片懒加载

标签（空格分隔）： js 优化

前言
图片懒加载是一个很实用的技术，图片的懒加载最常见的是只加载网页显示在屏幕中的图片，如果用户没有滚动到网页靠下的部分，那么那部分图片是不加载的。这可以有效的减少对服务器的请求，降低服务器的带宽，对用户的体验有很大幅度的提升。

实现
首先需要引入jQuery.js 和 jquery.lazyload.js
然后在需要懒加载的图片上的 src 的属性换成 data-original
例：<img class="lazy" data-original="img/插画/6.jpg" src="img/占位图.jpg" alt="插画出现错误" height="800px" width="800px">

而 src 属性中的图片换成占位符的图片（例如 1x1 像素的灰色图片或者 loading 的 gif 图片）
还需要添加 width 和height属性有助于在图片未加载时占满所需要的空间
然后调用lazyload方法 传递参数即可

$(function () {
$("img.lazy").lazyload({effect:"fadeIn"});
});
